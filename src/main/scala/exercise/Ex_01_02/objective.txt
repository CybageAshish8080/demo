Challenge
=======================

1. Create a new project in Scala IDE
2. Create a worksheet in this project
3. Write code for a payroll function
4. payroll function will calculate weekly paycheck for an hourly employee
	In our example,
	Pay hourly employees' overtime (1.5 times hourly rate)
	which means first 40 hours will calculate with regular rate.
	anything above 40 hours should be calculated 1.5 times of hourly rate.
5. Use an hourly rate of 10.50
6. Check if employee is hourly or salaried before calculating their paycheck
7. If they are salaried, just print a message
8. Test your program with several employees
	Hourly/20 hours
	Hourly/40 hours
	Hourly/50 hours
	Salaried
